This space management system leverages a robust Internet of Things (IoT) architecture, utilizing high-precision pressure sensors and force-sensitive resistors to dynamically monitor seat occupancy by transducing physical interactions into real-time digital telemetry. The acquired occupancy data is transmitted from the microcontroller units via secure network protocols for seamless integration with the SIX ITB information system's backend infrastructure. Facilitated by optimized API (Application Programming Interface) calls, this synchronization allows for the instantaneous visualization of live seating availability within the "Space" module, providing a comprehensive spatial information interface for the academic community to optimize campus facility utilization through a smart, transparent, and automated engineering framework.
